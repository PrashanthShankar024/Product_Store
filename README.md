# STYLE MANDIR - Premium Fashion Store

A modern, fully responsive e-commerce product store built with **HTML, CSS, and JavaScript** as a college/project assignment.

![STYLE MANDIR Fashion Store]

## ✨ Features

- **Beautiful Dark Modern UI** with glassmorphism effect
- **7 Product Categories**: Shirts, Pants, Jeans, Jackets, Hoodies, Kurtas, Sarees
- **Real-time Search** functionality
- **Category Filtering** with active button highlight
- **Sorting** (Price Low to High, High to Low, Most Liked)
- **Like / Wishlist** system with heart animation (saved in LocalStorage)
- **Working Shopping Cart** (Add, Increase/Decrease quantity, Remove)
- **Quick View Modal** for each product
- **Responsive Design** (Mobile + Desktop friendly)
- **Toast Notifications** when adding to cart
- **Data Persistence** using LocalStorage (likes & cart survive page refresh)
- **Smooth Animations & Hover Effects**

## 🛠 Technologies Used

- HTML5
- CSS3 (Custom + Bootstrap 5)
- JavaScript (Vanilla)
- Bootstrap 5 (for Modal & Offcanvas)
- Font Awesome Icons
- LocalStorage

## 🚀 How to Run the Project

1. Download or clone the project folder.
2. Make sure you have these 4 files in the same folder:
   - `index.html`
   - `style.css`
   - `script.js`
   - `README.md`
3. Double-click `index.html` to open it in your browser (Chrome recommended).
4. No server needed — works offline.
